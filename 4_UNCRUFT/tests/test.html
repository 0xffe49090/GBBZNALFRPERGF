URL Smoke Test Suite

Each link opens in a new tab. After navigation, compare the resulting address-bar URL with the expected cleaned URL shown below it.

1. Basic UTM removal

Removes utm_source and utm_medium, while retaining the functional id parameter.

Open basic UTM test

Expected:
https://example.com/article?id=42

2. Advertising click identifiers

Removes the Facebook fbclid and Google gclid identifiers, while preserving page=2.

Open click-ID test

Expected:
https://example.org/test?page=2

3. Tracking parameters mixed with functional data

Removes utm_campaign and utm_content, while retaining the unknown functional parameter token.

Open mixed-parameter test

Expected:
https://example.net/report?token=keep-me

4. Mailchimp tracking parameters

Removes mc_cid and mc_eid. The generic ref parameter should remain because it is not globally removed by the default configuration.

Open Mailchimp test

Expected:
https://example.com/?ref=keep-me

5. Repeated tracking parameter

Tests whether every occurrence of a repeated utm_source parameter is removed while preserving id=99.

Open repeated-parameter test

Expected:
https://example.org/path?id=99

6. Encoded value and URL fragment

Removes utm_source, while preserving the encoded search value and the #section-2 fragment.

Open encoding and fragment test

Expected:
https://example.net/docs?q=hello%20world#section-2

7. Negative control

Contains no configured tracking parameters. The extension should leave the URL unchanged.

Open unchanged control test

Expected:
https://example.com/search?q=security&page=3&id=123

8. YouTube domain-specific rule

Tests the bundled domain-specific rules. It should preserve the video identifier in v, while removing si and feature.

Open YouTube domain-rule test

Expected:
https://www.youtube.com/watch?v=dQw4w9WgXcQ

9. Large mixed tracking set

Exercises several independent tracking families in one request. Only id and mode should survive.

Open large mixed-set test

Expected:
https://example.com/demo?id=123&mode=test

10. Tracking-only query string

Every parameter is removable. The resulting URL should have no query string and should not retain a trailing question mark.

Open tracking-only test

Expected:
https://example.org/
